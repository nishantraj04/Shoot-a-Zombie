# Introduction

This is a simple shooting game, I made it from python to update my logical mind and tbh it is really fun to make games by yourself rather then doing the vibe coding.

# How to install and use this game

The main requirement is you must have Python installed in your system.

1) Clone this repo

```bash
git clone <repo-link>
```

2) Install pygame 

```bash
pip install pygame
```

3) Execute the `main.py` file

```bash
python ./main.py
```

Enjoy the comment, and feel free to update it or do experiment here, and also don't forget to share it with me.

# How to play this game?

There is no time limit, zombies are coming, shoot them all, remember for shooting, right click or tap the zombie to shoot them, remeber they will increase in specific levels and also you can only ignore 15 zombies, make high score!
